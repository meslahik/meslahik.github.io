Selected Projects

Image Segmentation and Object Detection     [OpenCV, Python], Winter 2015

  • Implemented several different image segmentation methods including Adaptive Thresholding, Otsu, and Watershed algorithms.

  • Developed an application that supports manual object detection by taking advantage of binarization, thresholding, and labling techniques.

  • Compared the methods by running them on a benchmark dataset.

Autonomous Robotic Path Planning Using RRT Algorithm     [Matlab], Summer 2014

  • Rapaidly-exploring Random Tree (RRT) is an algorithm to search high dimensional spaces. We utilized the algorithm for solving autonomous path planning problem in an environment with obstacles.

  • We used RRT*FN library for simulating the problem. The simulation results show high capability of the algorithm in solving the path planning problem.

Khepera Robot Simulation     [Matlab], Fall 2013

  • Developed a simulator for Khepera robot (a small differential wheeled mobile robot). The robot is supposed to navigate in different areas while avoids obstacles.

  • Utilized linear membership functions (a set of expert system rules) in conjunction with FIS variables to simulate behavior of Khepera Robot.

  • Studied the impact of velocity on the robot’ performance.

Hospital Information System     [C#, SQL], Winter 2012

  • Designed and developed a web-based Hospital Information System (HIS) by using ASP.Net, C# and Microsoft SQL Server.

  • The HIS system supports appointment booking, billing, and some other functionalities of HIS systems. It also generates overall reports automatically in every desired period of time.
