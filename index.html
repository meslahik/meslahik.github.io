Research

My research interests include:

  • Distributed Data Structures

  • Dependable Distributed Systems

Currently, I am working on impelemnting Paxos, an important consensus algorithm, on RDMA networks. For that, we are going to use RDMA verbs efficiently so maximizing the consensus implementation. Later, we will use this consensus leyer for implementing fast fault-tolarent systems.

Lately, I worked on a distributed B+Tree algorithm. We partition the tree by distributing its nodes in a number of partitions. The main challenge was to provide scalable performance when the number of partitions increases. We use a partitioned state machine protocol developed in our group before.
